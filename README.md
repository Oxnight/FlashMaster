# FlashMaster

FlashMaster est une application de bureau conçue pour créer, gérer et étudier des flashcards. Elle est développée en Python avec PyQt5 pour l'interface graphique.

## Fonctionnalités

- **Création et gestion de decks** : Organisez vos flashcards en decks.
- **Ajout et édition de cartes** : Ajoutez des questions et des réponses à vos cartes.
- **Session d'étude** : Étudiez vos cartes avec des statistiques de réussite.

## Installation

Pour installer et exécuter FlashMaster, suivez ces étapes :

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/Oxnight/FlashMaster.git
   cd FlashMaster
   ```
2. **Créer un environnement virtuel et l'activer** :
   ```bash
   python -m venv env
   source env/bin/activate
   ```
3. **Installer les dépendances** :
   ```bash
   pip install -r requirements.txt
   ```
4. **Lancer l'application** :
   ```bash
   python main.py
   ```
